Recruitment Portal Project

This is a python project where my responsibility is front-end development using HTML5 and Javascript.

Problem

The key problem areas with the current system is data redundency. This permits correction of errors in the stored data. Another problem was that the current system has no way of letting users know what tasks they have completed which leads to a repeatition of the same tasks.

Solution

The Recruitment portal will solve the problem by storing none redundant data in a mysql database, removing anomalies and making sure that the data is normalised. The portal also has a dasboard that will guide users over a sequence of tasks that need to be completed. It will clearly state which tasks have been completed and which are yet to be completed.
